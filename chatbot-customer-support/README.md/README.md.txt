# ?? Customer Support Chatbot

## ?? Project Overview
This project is a customer support chatbot built using Dialogflow.  
It is designed to handle common customer queries like greetings, order status, and refund-related questions.

## ??? Tools & Technologies
- Dialogflow (No-code chatbot builder)
- NLP-based intent recognition
- Optional integrations: Telegram / Web

## ? Features
- Greeting responses
- FAQ handling (Order status, Refund policy)
- Smart fallback responses
- 24/7 automated customer support

## ?? Testing
The chatbot was tested using Dialogflow's built-in simulator with multiple user queries.

Screenshots of intents and chatbot testing are included in the `screenshots/` folder.

## ?? Outcome
A functional AI-powered chatbot capable of improving customer support efficiency.
